## â–¶ï¸ How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Hadia-Sarwar/Compawnion
cd Compawnion
```

### 2. Install Dependencies

**Install frontend:**
```bash
cd frontend
npm install
```

**Install backend:**
```bash
cd ../backend
npm install
```

### 3. Setup Environment Variables

Create a `.env` file inside the **backend** folder:

```ini
MONGO_URI=your_mongodb_link
JWT_SECRET=your_secret
CLOUDINARY_KEY=optional_if_used
```

### 4. Run the Project

**Start backend:**
```bash
cd backend
npm start
```

**Start frontend:**
```bash
cd frontend
npm start
```
